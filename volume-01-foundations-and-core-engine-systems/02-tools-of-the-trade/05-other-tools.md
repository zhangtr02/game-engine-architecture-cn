### 2.5 其他工具

游戏程序员的工具箱中还有许多其他常用工具。这里我们不会深入介绍它们，但下面这份列表会让你知道它们的存在，并在你想了解更多时为你指明方向。

- *Difference tools.* 差异工具（difference tool），也称 diff 工具，是一种比较文本文件两个版本并判断它们之间发生了哪些变化的程序。（关于 diff 工具的讨论，见 [96]。）Diff 通常是逐行计算的，不过现代 diff 工具也可以显示某一行中被修改的一段字符。大多数版本控制系统都自带 diff 工具。有些程序员喜欢某个特定的 diff 工具，并会配置自己的版本控制软件使用他们选择的工具。流行工具包括 ExamDiff [97]、AraxisMerge [98]、WinDiff（可在大多数 Windows 版本的 Options Packs 中获得，也可从许多独立网站获得）、GNU diff tools package [99]，以及我个人最喜欢的 Scooter Software 出品的 Beyond Compare [100]。

- *Three-way merge tools.* 当两个人编辑同一个文件时，会产生两组彼此独立的 diff。能够把两组 diff 合并成文件最终版本，并同时包含两个人修改内容的工具，称为三方合并工具（three-way merge tool）。“三方”这个名称指的是该过程涉及文件的三个版本：原始版本、用户 A 的版本和用户 B 的版本。（关于双向合并和三方合并技术的讨论，见 [101]。）许多合并工具都带有配套的 diff 工具。一些流行的合并工具包括 AraxisMerge [98] 和 WinMerge [102]。Perforce 也自带一个优秀的三方合并工具 [103]。

- *Hex editors.* 十六进制编辑器（hex editor）是一种用于检查和修改二进制文件内容的程序。数据通常以十六进制格式的整数显示，因此得名。大多数优秀的十六进制编辑器都可以把数据显示为 1 字节到 16 字节不等的整数、32 位和 64 位浮点格式，以及 ASCII 文本。在追踪二进制文件格式问题，或者逆向工程某个未知二进制格式时，十六进制编辑器尤其有用——这两件事在游戏引擎开发圈中都相当常见。市面上的十六进制编辑器多得数不胜数；我使用 Expert Commercial Software 出品的 HexEdit [104] 效果不错，但你的体验可能会有所不同。

作为游戏引擎程序员，你无疑还会遇到其他能让工作更轻松的工具。不过，我希望本章已经覆盖了你日常最常用的主要工具。
