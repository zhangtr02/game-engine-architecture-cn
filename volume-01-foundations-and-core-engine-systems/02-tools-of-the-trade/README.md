# 2 Tools of the Trade

在我们踏上游戏引擎架构这片迷人领域的旅程之前，先为自己配备一些基本工具和补给是很重要的。在接下来的两章中，我们将回顾这段旅程所需的软件工程概念与实践。在第 2 章中，我们将探索大多数专业游戏工程师使用的工具。随后在第 3 章中，我们会通过回顾面向对象编程、设计模式和大规模 C++ 编程领域中的一些关键主题，完成出发前的准备工作。

游戏开发是软件工程中要求最高、覆盖范围最广的领域之一。因此请相信我，如果我们要安全穿越接下来将要介绍的这片有时颇为险峻的地形，就需要装备齐全。对于一些读者来说，本章以及下一章的内容会非常熟悉。不过，我仍然鼓励你不要完全跳过这两章。我希望它们能成为一次愉快的复习；而且谁知道呢，你也许还能学到一两个新技巧。
