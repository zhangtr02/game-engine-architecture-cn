## 5.1 用二维解决三维问题

在接下来这一章中，我们将要学习的许多数学操作，在 2D 和 3D 中都同样适用。这是个非常好的消息，因为它意味着你有时可以通过在 2D 中思考和画图，来解决一个 3D 向量问题（这要容易得多！）。遗憾的是，2D 与 3D 之间的这种等价性并不总是成立。有些操作，例如叉积，只在 3D 中有定义；有些问题也只有在同时考虑三个维度时才有意义。

尽管如此，从手头问题的简化二维版本开始思考，几乎从来都不会有什么坏处。一旦你理解了 2D 中的解法，就可以进一步思考这个问题如何扩展到三维。在某些情况下，你会很高兴地发现，你的 2D 结果同样适用于 3D。在另一些情况下，你也能够找到一个坐标系，使问题实际上 **就是** 二维的。在本书中，只要 2D 和 3D 之间的区别无关紧要，我们都会使用二维图示。
