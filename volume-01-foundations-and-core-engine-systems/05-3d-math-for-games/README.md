# 5 3D 游戏数学

游戏是在某种计算机上实时模拟虚拟世界的数学模型。因此，数学渗透在游戏行业中我们所做的一切事情之中。游戏程序员几乎会用到数学的所有分支，从三角学到代数，从统计学到微积分。然而，作为游戏程序员，你最常用到的数学类型，远远要数 3D 向量和矩阵数学，也就是 3D **线性代数**（linear algebra）。

即使只是数学中的这一分支，也非常宽广而深入，因此我们不可能指望在单独一章中对它进行很深入的覆盖。相反，我会尝试概述典型游戏程序员所需的数学工具。在此过程中，我也会提供一些技巧和窍门，帮助你把这些相当容易混淆的概念和规则理清楚。关于游戏 3D 数学的优秀深入讲解，我强烈推荐 Eric Lengyel 关于该主题的书籍 [36] 和 [34]。Christer Ericson 关于实时碰撞检测的书 [15] 中的第 3 章，也是一个非常优秀的资源。
