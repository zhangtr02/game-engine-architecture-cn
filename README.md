# Game Engine Architecture 中文版翻译

本项目用于记录《Game Engine Architecture》第四版两卷本的中文翻译与阅读整理。

## 原书信息

- 书名：Game Engine Architecture
- 版本：Fourth Edition
- 作者：Jason Gregory
- 卷一：Volume I: Foundations and Core Engine Systems
- 卷二：Volume II: Graphics, Motion, and Sound

## 项目说明

本项目基于英文原版内容整理中文翻译，主要用于个人学习、技术理解与阅读记录，不用于任何商业用途。

翻译过程使用 AI 辅助完成，并会根据个人理解对术语、表达和排版进行人工校对与整理。部分章节仍可能存在译法不统一、表达生硬或图片说明待校对的情况。

如本项目内容涉及版权问题或存在侵权之处，请联系本人，我将及时删除或修改相关内容。
