# 18 你是说还有更多

恭喜！你已经完成了这段穿越游戏引擎架构版图的旅程，而且毫发无损地抵达了终点（至少希望没有比出发时更糟）。如果运气不错，你已经对构成典型游戏引擎的主要组件有了相当多的了解。不过，当然，一个旅程的结束，正是另一个旅程的开始。本书所涵盖的每一个主题，还有大量内容值得继续学习。随着技术和计算硬件不断进步，游戏中将会出现更多可能性，也会有更多引擎系统被发明出来以支持这些可能性。更重要的是，本书的重点一直放在游戏引擎本身。我们甚至还没有真正开始讨论内容丰富的**游戏玩法编程**世界，而这个主题本身就足以写成许多卷书。

在接下来的几个简短小节中，我会列出一些本书没有足够篇幅深入讨论的引擎系统和游戏玩法系统，并为希望进一步了解这些内容的读者推荐一些资源。
