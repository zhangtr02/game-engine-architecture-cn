# 13 动画系统

现代 3D 游戏大多围绕**角色**展开——这些角色通常是人类或类人角色，有时也可能是动物或外星人。角色之所以特殊，是因为它们需要以流畅、自然的方式运动。相比模拟和动画化车辆、抛射物、足球、俄罗斯方块方块等刚体对象，这会带来一系列全新的技术挑战。让角色拥有自然可信的动作，这项任务由引擎中的一个组件负责，它被称为**角色动画系统**（character animation system）。

正如我们将会看到的那样，动画系统为游戏设计师提供了一套强大的工具集，这些工具不仅可以用于角色，也可以用于非角色对象。任何并非 100% 刚性的游戏对象，都可以从动画系统中受益。因此，当你在游戏中看到一辆带有运动部件的载具、一台带有关节结构的机械、微风中轻轻摇曳的树木，甚至是一栋正在爆炸的建筑时，很可能这个对象至少部分使用了游戏引擎的动画系统。
