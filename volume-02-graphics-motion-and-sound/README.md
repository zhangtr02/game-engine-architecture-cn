# Game Engine Architecture

在这部全新且经过改进的第四版 *Game Engine Architecture* 中，Jason Gregory 凭借其在 Midway、Electronic Arts 和 Naughty Dog 二十年的从业经验，对游戏引擎软件开发的理论与实践进行了系统介绍。在本书中，AAA 游戏工作室所使用的各种技术与方法都会得到详细解释，并且它们在真正工业级游戏引擎中的作用也会得到说明。

第四版延续了前几版对游戏引擎架构的全面覆盖，并且本卷重点介绍构成每一个游戏引擎核心的基本概念、技术、工具和引擎系统。

本书旨在作为一本入门教材，但它也能为有经验的游戏程序员提供有益视角，帮助他们理解那些自己可能尚未有深入经验的游戏开发技术领域。和以往一样，本版也提供了大量参考文献与引用资料，因此对于希望深入研究游戏开发流程某一特定方面的读者来说，本书也是一个极佳的起点。

## Key Features

- 覆盖游戏引擎软件开发的理论与实践
- 示例以具体技术为基础，但讨论范围并不局限于某个特定引擎或 API
- 包含所需的全部数学背景知识
- 既适合初学者系统阅读，也包含适合资深工程师的内容

## About the Author

Jason Gregory 自 1999 年 3 月起一直在游戏行业担任软件工程师，并自 1994 年起从事专业软件工程工作。他的游戏编程生涯始于位于圣迭戈的 Midway Home Entertainment，在那里他为街机游戏 *Hydro Thunder™ 2* 开发工具、引擎技术和游戏玩法代码。他还为 *Freaky Flyers™* 和 *Crank the Weasel* 编写了 PlayStation® 2 / Xbox 动画系统。

2003 年，Jason 加入位于洛杉矶的 Electronic Arts，参与 *Medal of Honor™ Pacific Assault* 的引擎与游戏玩法技术开发，并担任 *Medal of Honor:™ Airborne* 项目的首席工程师。Jason 目前是 Naughty Dog Inc. 的首席程序员，他和他的团队正在为 PlayStation® 5 开发一款令人期待的新游戏。他还曾为 Naughty Dog 的多部作品开发引擎与游戏玩法软件，包括 *Uncharted: Drake’s Fortune™*、*Uncharted 2: Among Thieves™*、*Uncharted 3: Drake’s Deception™*、*Uncharted 4: A Thief’s End™*、*Uncharted: The Lost Legacy™*、*The Last of Us Part I™* 和 *The Last of Us Part II™*，这些作品面向 PlayStation® 5、PlayStation® 4、PlayStation® 3 和 PC 平台。Jason 还曾在南加州大学开发并教授游戏技术课程。
