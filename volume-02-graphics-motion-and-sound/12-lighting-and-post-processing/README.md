# 12 光照与后处理

在当今游戏产业中，玩家渴望高度的视觉保真度。当然，也有一些例外，比如有些游戏会有意追求卡通化或“复古”的外观；但如今大多数 3A 游戏都旨在实现接近照片级真实感的效果。那么，是什么让一个优秀的 3D 场景区别于一个看起来平淡无奇的场景呢？几何细节当然起着重要作用——如果我们要相信自己看到的是一张真实的人脸，那么一旦观察者能够察觉三角形之间的硬边，这种幻觉就会被打破。动态效果同样起着重要作用——一个静态场景通常会比包含大量以自然方式运动的物体的场景显得更不真实。然而，可以说，在实现照片级真实感渲染场景时，最重要的因素是**光照**（lighting）。在 Chapter 11 中，我们奠定了 3D 渲染技术的基础，并探讨了渲染场景的实用方法。在本章中，我们将通过深入光照这个迷人的世界，补全对 3D 图形的讨论。
